Paragraph

Paragraph represents a consecutive lines of non-special lines.

Hello World.

Yields:

{
  "type": "paragraph",
  "children": [
    {
      "type": "text",
      "value": "Hello World."
    }
  ]
}

Section

Section starts with a headline, and it can be nested.

* Headline 1
And this is the content.
** Headline 1.1
*** Headline 1.1.1
* Headline 2

Yields:

{
  "type": "section",
  "children": [
    {
      "type": "headline",
      "children": [
        {
          "type": "text",
          "value": "Headline 1"
        }
      ],
      "level": 1
    },
    {
      "type": "paragraph",
      "children": [
        {
          "type": "text",
          "value": "And this is the content."
        }
      ]
    },
    {
      "type": "section",
      "children": [
        {
          "type": "headline",
          "children": [
            {
              "type": "text",
              "value": "Headline 1.1"
            }
          ],
          "level": 2
        },
        {
          "type": "section",
          "children": [
            {
              "type": "headline",
              "children": [
                {
                  "type": "text",
                  "value": "Headline 1.1.1"

              ],
              "level": 3
            }
          ],
          "level": 3
        }
      ],
      "level": 2

  ],
  "level": 1
}

Headline

Headline can have keyword, priority, tags, planning, drawers.

* TODO [#A] Hello World   :tag1:tag2:
DEADLINE: <2018-01-01 Mon>
:PROPERTY:
key0: value0
key1: value1
:END:

Yields:

{
  "type": "section",
  "children": [
    {
      "type": "headline",
      "children": [
        {
          "type": "text",
          "value": "Hello World"
        },
        {
          "type": "planning",
          "keyword": "DEADLINE",
          "timestamp": "<2018-01-01 Mon>"
        },
        {
          "type": "drawer",
          "name": "PROPERTY",
          "value": "key0: value0\nkey1: value1"
        }
      ],
      "level": 1,
      "keyword": "TODO",
      "priority": "A",
      "tags": [
        "tag1",
        "tag2"
      ]
    }
  ],
  "level": 1
}
